┌───────────────────────────────────────────────────────────────┐
│  TROUBLESHOOTING                                              │
└───────────────────────────────────────────────────────────────┘

Backend won't start:
  • Check: uv sync
  • Verify: .env file with CMC_API_KEY
  • Check: Port 8000 available

Frontend won't start:
  • Run: cd frontend && npm install
  • Check: Port 5173 available
  • Verify: Node.js 18+

No real-time updates:
  • Check backend is running
  • Open DevTools > Network > EventStream
  • Verify SSE connection to /api/v1/stream
